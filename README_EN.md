# go-json-datetime

Conveniently process golang json `time.Time`

## Install

`go get -u github.com/reatang/go-json-datetime`

## Time type

- codec YYYY-MM-dd hh:mm:ss ：`jsondt.DateTime`
- codec YYYY-MM-dd ：`jsondt.Date`
- codec hh:mm:ss ：`jsondt.Time`
